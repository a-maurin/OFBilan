# Tables de référence

Ce dossier regroupe les tables de référence versionnées utilisées par les bilans.

## Contenu

- correspondances thèmes/domaines/usagers ;
- nomenclatures (dont NATINF) ;
- tables métier de normalisation.

## Règles d’usage

- ce dossier est réservé aux référentiels stables ;
- les données opérationnelles sont placées dans `data/sources/` ;
- les sorties sont produites dans `data/out/`.

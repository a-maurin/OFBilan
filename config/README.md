# Configurations du projet

Ce dossier centralise les **configurations versionnées** du projet :

- `profils_bilan/` : profils YAML des bilans thématiques (agrainage, chasse, piégeage, types d'usagers, procédures, etc.) ;
- `cartes/` : configuration du générateur de cartes (profils de cartes, symbologies) ;
- fichiers CSV/JSON de référence (thèmes, listes de NATINF, types d'usagers, etc.).

Les données sources volumineuses restent dans `sources/` ou `data/` (non versionnés) et les sorties générées dans `out/` (non versionné).
